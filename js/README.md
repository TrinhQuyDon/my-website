# Hồ sơ Năng lực - Trịnh Quý Đôn

Đây là dự án website hồ sơ cá nhân (Portfolio) được xây dựng bằng ngôn ngữ HTML5 thuần tuý, thuộc khuôn khổ bài tập Lab 2. 

## Cấu trúc dự án
* `index.html`: Trang chủ giới thiệu tổng quan.
* `about.html`: Trang chi tiết về kỹ năng, lộ trình học tập và môn học.
* `contact.html`: Trang biểu mẫu liên hệ có tích hợp HTML5 Validation.

## Tính năng nổi bật
* Giao diện Responsive đơn giản.
* Áp dụng đầy đủ các thẻ Semantic HTML5 (`<header>`, `<main>`, `<section>`, `<footer>`, `<figure>`).
* Form liên hệ hoàn chỉnh với các ràng buộc dữ liệu (Required, Pattern, Minlength).
* Tối ưu SEO cơ bản với thẻ `Title`, `Meta Description` và Open Graph tags.
